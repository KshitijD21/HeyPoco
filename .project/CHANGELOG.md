# HeyPoco — Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] — 2026-02-22

### Added
- Initial project scaffold
- FastAPI backend with OpenAI integration (Whisper, GPT-4o, embeddings)
- Next.js 14 frontend with App Router
- Supabase auth (login/signup) + RLS policies
- Voice recording with MediaRecorder API
- AI-powered entry extraction and confirmation flow
- Entry feed with date grouping and category filtering
- Conversational Q&A (RAG pipeline)
- Supabase schema with vector search support
- Project tracking folder (`.project/`)
